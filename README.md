# SharpOffice

Convert office document to pdf and swf

## Installation

Add this line to your application's Gemfile:

    gem 'sharp_office'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sharp_office

## Usage

SharpOffice.process('/path/to/document')
